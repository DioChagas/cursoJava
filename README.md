### Primeiros passos
* [x] Leitura de dados de entrada (*System.in*)
* [x] Operadores aritméticos
* [x] Operadores Lógicos
* [x] Comandos condicionais (*if, else, Switch case*)
* [x] Estrutura de repetição (*for, While, do while*)
